## Описание лабораторной работы

В данной лабораторной работе реализуются две классические задачи программирования на языке Java:

1. **Поиск простых чисел** в диапазоне от 2 до 100.
2. **Определение палиндромов** — проверка, читается ли слово одинаково слева направо и справа налево.

Основная цель — закрепить навыки работы с циклами, условными операторами, строковыми методами и декомпозицией кода (разбиение на вспомогательные методы).

---

## Задание 1: Поиск простых чисел

### Описание

Программа находит и выводит все простые числа в диапазоне от 2 до 100. Простое число — это натуральное число, которое делится без остатка только на 1 и на само себя.

### Подробное объяснение кода

Весь код организован в одном классе `Laba_1_Task_1`, который содержит два метода:

#### Метод `main(String[] args)`

- Запускает цикл `for` от `i = 2` до `i <= 100`.
- Для каждого значения `i` вызывает вспомогательный метод `isPrime(i)`.
- Если метод возвращает `true` — число выводится на экран через `System.out.print()` с пробелом.
- После завершения цикла выполняется перевод строки для аккуратного вывода.

#### Метод `isPrime(int n)`

- Принимает целое число `n` и проверяет, является ли оно простым.
- В цикле `for` перебирает возможные делители `j` от `2` до `n-1`.
- Если при каком-то `j` остаток от деления `n % j` равен `0`, значит, найден делитель — число не простое, метод сразу возвращает `false`.
- Если цикл завершился без нахождения делителей, возвращается `true`.


## Задание 2: Определение палиндромов

### Описание

Программа проверяет строки (слова) на принадлежность к палиндромам — словам, которые читаются одинаково в обоих направлениях. Например: «шалаш», «радар», «потоп». Регистр букв не учитывается.

### Подробное объяснение кода

Код организован в классе `Laba_1_Task_2` и содержит три метода.

#### Метод `main(String[] args)`

- Сначала проверяется, были ли переданы аргументы командной строки (`args.length == 0`).
- Если аргументов нет — создаётся тестовый массив строк по умолчанию: `{"шалаш", "Шалаш", "потоп", "дерево", "радар", "код"}`.
- Запускается цикл `for` по всем элементам массива.
- Для каждой строки `s` вызывается метод `isPalindrome(s)`.
- В зависимости от результата выводится сообщение: «является палиндромом» или «не является палиндромом».

#### Метод `reverseString(String s)`

- Этот метод переворачивает строку.
- Создаётся пустая строка `reversed`.
- Цикл `for` идёт от последнего индекса строки (`s.length() - 1`) до первого (0).
- На каждой итерации текущий символ `s.charAt(i)` добавляется в `reversed`.
- В конце метод возвращает перевёрнутую строку.

Пример: на входе «код» → на выходе «док».

#### Метод `isPalindrome(String s)`

- Принимает строку `s`.
- Сначала приводит её к нижнему регистру с помощью `s.toLowerCase()`. Это нужно, чтобы «Шалаш» и «шалаш» считались палиндромами одинаково (регистр не влияет).
- Затем вызывает `reverseString()` для получения перевёрнутой версии строки в нижнем регистре.
- Сравнивает исходную строку в нижнем регистре с перевёрнутой с помощью `equals()`.
- Если они равны — возвращает `true` (палиндром), иначе — `false`.
